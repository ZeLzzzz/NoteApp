# Usage

**1. Download Source Code**

```bash
 git clone https://github.com/ZeLzzzz/NoteApp.git
 cd NoteApp
```

> OR

- [Download Zip File](https://github.com/ZeLzzzz/NoteApp/archive/refs/heads/main.zip)

**2. Migrate**

```artisan migrate
 php artisan migrate --seed
```

**3. Run the program**

```artisan serve
 php artisan serve
```
```npm run dev
 npm run dev
```
